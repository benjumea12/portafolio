TidBok


Volver

TidBok es un proyecto personal que surgió de mi experiencia entrenando ciclismo. Es una aplicación que le permitirá a los deportistas y entrenadores llevar un registro de tiempos y progresos con un diseño moderno y con funciones especificas para cumplir su propósito en deportes como el ciclismo y el atletismo.

La proyecto se encuentra en etapa de diseño y desarrollo pero mi objetivo es convertirla en un emprendimiento a futuro y poderla sacar al mercado.


Descripción

TidBok le va a permitir al los usuarios medir su tiempo y progreso en sus recorridos favoritos, generar gráficas y estadísticas que le permitan saber como avanza su entrenamiento, también le permitirá agregar otros deportistas y crear un equipo donde se puedan competir entre ellos.

Deasarrollo

El proyecto esta dividido en dos partes, una aplicación web y una aplicación móvil que se complementan, la aplicación web se va a desarrollar con React js, la aplicacion movil se va a desarrollar con React Native ambas utilizando componentes UI propios, y el backend se va a desarrollar con Node js.

Actualmente hay un prototipo desarrollado de la aplicación web desarrollada en la ultima version de React-Hooks y React-TypeScrip, utiliza componentes UI y animaciones propias desarrollados con la librería de administración de estilos Styled-Components sin utilizar componentes UI externos.
