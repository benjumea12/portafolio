TidBok


arrow_back Volver

Tidbok es un proyecto personal que surgio de mi experiencia entrenando ciclismo, es una aplicación que le permitirá a los deportistas que entrenen a un nivel competitivo medir tiempos de entrenamiento y su progreso. Una aplicación deportiva con un diseño moderno y minimalista, con funciones especificas para cumplir su propósito.


La etapa de diseño de la aplicación web ya finalizo y la etapa de diseño de la aplicación móvil esta en curso, se esta utilizando las herramientas de Adobe XD y Adobe Illustrator para el diseño.

La aplicación web esta en etapa de desarrollo.

Tecnologias: Adobe XD, Adobe Illustrator, React js, material UI, React Native, Native base, Node js, Express js, mongo DB
