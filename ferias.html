Ferias Comerciales


Volver

Ferias Comerciales es un proyecto educativo desarrollado por un grupo de estudiantes del tecnologo ADSI (análisis y desarrollo de sistemas informáticos) del SENA, es la la solución a un problema que tenían los estudiantes y el SENA en el proceso de desarrollo de una feria, desde su planeación hasta su terminación y entrega de resultados.


Descripción

Ferias Comerciales le permite a los directivos del SENA planear un feria con anticipación, los encargados de realizar la feria podrán agregar los participantes con sus stands y  al momento de realizar la feria darles acceso a un sistema de  facturación, también podrán generar informes del estado de sus ventas.

Deasarrollo

El proyecto consta de una aplicación web en la cual realizamos todo el proceso de desarrollo de software desde la recolección de requerimiento hasta su presentación, fue desarrollada con el lenguaje PHP utilizando el framework CodeIgniter y como base de datos utilizamos SQL.

Actualmente hay un prototipo desarrollado de la aplicación web que utiliza componentes UI propios desarrollados con la librería de administración de estilos Styled-Components sin utilizar componentes UI externos.
